##Usage Instructions







##Installation



1. Upload Font Icons to the `/wp-content/plugins/` directory.



2. Activate the plugin through the 'Plugins' menu in WordPress.



3. go to woocomerce -> settings -> shiping -> LP EXPRESS-PA
